## Description

[Jump](https://app.ajump.od.ua/) API is a backend application based on nestjs. For Jump mobile and web apps.

## Installation

```bash
$ npm install
```

## Building the app

```bash
# builds application
npm run build
```

## Starting the app

```bash
# development mode
npm run start:dev

# production mode
npm run start:prod
```

## Linter

```bash
# lint with --fix
npm run lint
```
